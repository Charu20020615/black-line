# My React App

## Scripts
- **dev**: `npm run dev`
- **build**: `npm run build`
- **preview**: `npm run preview`

## Setup
1. cd `my-react-app`
2. `npm install`
3. copy `.env.example` to `.env` and set `VITE_API_BASE_URL`
4. `npm run dev`

## Structure
Follows the requested layout with `api`, `assets`, `components`, `contexts`, `hooks`, `layouts`, `pages`, `redux`, `routes`, and `utils`.
